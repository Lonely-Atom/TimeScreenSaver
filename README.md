# TimeScreenSaver

## 时间屏保

### 使用 `.Net 6.0` 框架开发 `Winform` 程序

---

## 使用说明

### 快捷键

1. 切换小时制模式（12小时制模式/24小时制模式）
   - `Ctrl + W`
2. 切换主题模式（单色模式/多色模式）
   - `Ctrl + Q`
3. 切换上一个主题
   - `Shift + Tab`
   - `Ctrl + Shift + Tab`
4. 切换下一个主题
   - `Tab`
   - `Ctrl + Tab`
5. 复制时间
   - `Ctrl + C`
6. 截图
   - `Ctrl + X`
7. 暂停
   - `Space`
   - `Ctrl + Space`
8. 全屏
   - `F11`
9. 置顶窗口
    - `Ctrl + T`
10. 背景透明
    - `Ctrl + J`
11. 鼠标穿透
    - `Ctrl + K`
12. 锁定窗口（禁止缩放、移动、切换主题模式、切换主题、切换全屏、置顶窗口、收到托盘以及关闭）
    - `Ctrl + L`
13. 收到托盘
    - `Ctrl + M`
14. 退出（全屏状态为退出全屏，非全屏状态为关闭程序）
    - `ESC`
15. 关闭
    - `Alt + F4`

### 按键 + 鼠标滚轮

1. `Ctrl + 鼠标滚轮`：缩放字体大小
2. `Shift + 鼠标滚轮`：缩放窗体大小
3. `Alt + 鼠标滚轮`：调整窗体透明度

### 鼠标操作

1. `左键单击托盘`：收到托盘
2. `右键单击托盘`：菜单栏
3. `右键单击窗口`：菜单栏
4. `左键双击窗口`：全屏
5. `缩放窗口`：非全屏状态下，可鼠标左键拖拽窗口边缘缩放。
6. `移动窗口`：非全屏状态下，可鼠标左键按住窗口拖拽移动窗口。
